To build the webpages:

   make html


To automatically generate the API docs:

   sphinx-apidoc -f -M -o docs/source/ . lm_combustion radhydro

  (-M puts the module before submodule in the API pages; the directories after . are excludes))


To support notebooks:

pip3 install nbsphinx --user
dnf install pandoc
dnf install python3-ipython-sphinx

For numpy doc format:

pip3 install numpydoc --user


For the theme:

pip install sphinx_rtd_theme
